Завдання 2. Кнопка ScrollTop

1. Створіть веб-сторінку висотою більше, ніж 2 екрана (можна скопіювати код під завданням)

2. Добавте кнопку зі стрілкою вгору, стилями зафіксуйте її в нижній правій частині екрану.

3. Напишіть скрипт, що буде прокручувати сторінку на початок.

Рекомендації:
jQuery-функція .scrollTop(0) прокручує сторінку моментально.
Для плавної прокрутки можна використати функцію анімації .animate({scrollTop: 0},500).
Прокрутку необхідно застосовувати до тегів html та body (для різних браузерів).

4. Зробіть, щоб кнопка прокрутки з’являлась тільки коли користувач прокрутить сайт більше, ніж на екран вниз, ховайте, коли сайт прокрутиться вгору.

5. Переробіть код так, щоб кнопка створювалася динамічно, скриптом, тільки на тих сторінках, для <body> яких задано клас 'scrolltop'.

Завдання 3. Рухливий фон.

1. Зробіть сторінку-мотиватор: має займати однин екран (не повинно бути полос прокруток), по центру - крупний напис з 1-3 слів, під ним - речення-пояснення, слоган.

2. Добавте фонове зображення. При рухах миші фон повинен рухатися з невеликою швидкістю у протилежну сторону.

Рекомендації:
висоту сторінки можна задати або скриптом, або стилями, вказавши у якості одиниць виміру vh.
Фонове зображення можна зробити різними способами: у вигляді картинки <img>, фонового зображення для блока чи псевдоелемента.
Рухати це зображення можна змінюючи значення margin-top та margin-left, left та top, background-position.

3. Зробіть очі, що слідкують за курсором миші.

Контрольне завдання. Слайдер.

Технічне завдання.

1. На сторінці може розміщуватися 1 або більше слайдерів. Слайдер може складатися з блоків або бути у вигляді списку:

<div class="carusel">
  <div><img src="..." alt="..."></div>
  <div><img src="..." alt="..."></div>
  <div><img src="..." alt="..."></div>
  ...
</div>

<ul class="cycle-gallery">
  <li>Slide 1. Slogan.</li>
  <li>Slide 2. New slogan.</li>
  <li>Slide 3. Another slogan.</li>
  ...
</ul>

2. Слайди повинні змінюватися циклічно з проміжком 3 секунди (по замовчуванню). Ефект зміни - через прозорість (fade) чи зсув слайда вліво.

3. Добавити кнопки навігації "Наступний слайд", "Попередній слайд". Слайдер не повинен глючити під час ручного перемикання (коли час автоматичного і ручного перемикання співпадають - можливі конфлікти, смикання анімації, подвоєння класів активного слайда).

4. Добавити кнопки посторінкової навігації: кружечки у такій же кількості, як і кількість слайдів. Активний кружечок повинен виділятися стилем. При кліку на елемент навігації слайдер повинен переключатися на відповідний слайд.

5. Слайдер зробити у вигляді компонента jQuery, тобто, він має підключатися методом jQuery: $('.carousel').mySuperSlider(); винести код слайдера в окремий js-файл.

6. Зробити підтримку параметрів, щоб можна було задавати швидкість слайдера, тип анімації, швидкість перемикання:
$('.carousel').mySuperSlider({ timeout: 3000, type: 'shiftleft', speed: 1000 });

7. Добавити події: ініціалізація слайдера, початок та кінець анімації, ручне переключення на попередній, наступний слайди та слайд за номером.

8. Добавити користувацькі функції (в файл scripts.js), які б реагували на події: виводили в консоль назву події, id чи номер слайдера, якщо в слайді є зображення, то під слайдером виводити мініатюрку зображення 120x90 px, а також текст зі слайдера.

$('#slider2').on('next', function(){
  // [thumbnail]
  // текст слайдера
  // консоль: користувач натиснув кнопку Next в слайдері №2
});
