Заняття 19. Регулярні вирази
(regExp, regular expression).

Приклад регулярного виразу: /[^(\w)|(\@)|(\.)|(\-)]/ - проста (не універсальна) перевірка на коректність адреси e-mail.

Регулярний вираз - це строка-шаблон для пошуку у тексті.

Регулярні вирази використовуються для пошуку у всіх сучасних IDE, у текстових редакторах (наприклад, Notepad++), файлових менеджерах (Total/Double Commander).

Практично всі сучасні високорівневі мови програмування мають засоби для роботи з регулярними виразами.

Регулярні вирази широко застосовуються при парсингу інформації - виокремленні потрібних рядків даних з великих масивів інформації.

Основні конструкції:

. - шаблон будь-якого символа: /He..o/ == Hello, Hesko, He_5o.
.? - шаблон будь-якого символа чи його відсутність: /He.?.?o/ == Hello, Hemo, Heo.
.+ - шаблон декількох символів: 1 чи більше: /<p.+class/ == <p class, <p id="a" class="".
.* - шаблон декількох символів: 0 чи більше: /<p.*class/ == <pclass...
\. - шаблон символа крапки.
[abck] - шаблон одного символа з набору: /He[klm]lo/

В мові JavaScript у строкових змінних є декілька методів для роботи з регулярними виразами. Наступні методи в якості аргумента можуть приймати не тільки фіксовану строку, але і строку-шаблон - регулярний вираз:
var s = строка;
s.split(sep); - розбиття строки по сепаратору sep;
s.search(str); - пошук позиції входження підстроки;
s.match(str); - пошук всіх підстрок;
s.replace(str, newSubStr); - заміна однієї підстроки іншою

В JavaScript є спеціальний об’єкт для роботи з регулярними виразами: http://javascript.ru/RegExp. За цим посиланням добре описаний синтаксис побудови регулярних виразів.

Також варто знати, що бувають жадібний та лінивий пошук. Жадібний пошук намагається знайти максимально велику строку, лінивий - максимально коротку: wikipedia: Квантификация.

Завдання.

1. Знайдіть у коді сторінки всі <li>, у яких є клас bad (використовуючи вашу IDE та регулярні вирази).

2. Сервіс видає строку "На склад завезено 240 кг гарбузів". Зробіть парсінг цієї строки на вагу гарбузів, виведіть в консоль вагу гарбузів та номер позиції початку числа у строці.

3. Створіть input[type="text"]. В поле введіть будь-яке ім’я класу, наприклад, ".green-button". По кліку на кнопку поруч перевіряйте, чи назва класу є коректною, результат виводьте в консоль.

4. Створіть input[type="text"]. Поруч створіть span з жовтим фоном. В поле введіть адресу e-mail. При наведенні миші на span перевіряйте коректність адреси та виводьте у цей span результат червоним чи зеленим кольором в залежності від результату.

5. "З пункта А в пункт Б виїхав автомобіль зі швидкістю 90 км/год, подолав шлях за 2 години і витратив 12,5 л. пального". Створіть регулярний вираз, який поверне масив зі всіх числових даних задачі разом з їх одиницями виміру, результат вивести в консоль: ['90 км/год', '2 години', '12,5 л.']. Увага! Масив має повертати сам регулярний вираз, а не функція, яка може декілька раз застосувати цей вираз до строки.

/\+\d[-\s]?[(-\s]?\d{3}[)-\s]?[-\s]?\d{3}[-\s]?\d{2}[-\s]?\d{2}/g
