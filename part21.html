Заняття 21. Модифікація DOM в jQuery.

$('p').before('<div>...</div>'); - вставляє HTML-код чи елемент перед кожним вибраним елементом.

$('p').after('<div>...</div>'); - вставляє HTML-код чи елемент після кожного вибраного елемента.

$('p').prepend('<span>...</span>'); - вставляє HTML-код чи елемент всередину кожного вибраного елемента на початок.

$('p').append('<span>...</span>'); - вставляє HTML-код чи елемент всередину кожного вибраного елемента в кінець.

var target = $('#userpic');
$('em').prependTo(target).appendTo(target).insertBefore(target).insertAfter(target); - елемент чи HTML-код добавити до цілі - target.

$('#userpic').clone(); - клонувати елемент (у пам’ять) для його подальшого використання.

$('li').remove(); - видаляє вибрані елементи.

$('li').detach(); - видаляє вибрані елементи.

$('ul').empty(); - видаляє всі дочірні елементи (очищає вибрані елементи).

$('ul').wrap('<div class="wrap"></div>'); - робить обгортку довкола кожного з вибраних елементів.

$('p').wrapAll('<div class="wrap" />'); - робить одну спільну обгортку для всіх вибраних елементів.

$('p').wrapInner('<span />'); - робить обгортку для контента кожного вибраного елемента (огортає всередині).

$('p').unwrap(); - видаляє обгортку (безпосереднього батька).
