display: flex

Flexbox (flexible box, гнучкий блок) - новий підхід до розмітки блоків, який вдало поєднує табличне і блочне форматування.

Flexbox - набір стилів, які необхідно задавати як батьківським елементам, так і дочірнім.

Основа флексів - напрямок потоку. Блочні елементи розташовуються вертикально, строчні елементи - горизонтально, а флекс-елементам можна задавати напрямок розташування, в тому числі і зворотній: справа наліво, знизу вгору.

Напрямки потоків флекс-бокса.

Батьківський елемент

В першу чергу створимо батьківський елемент, обгортку:

<div class="wrap">
  ...
</div>

.wrap { display: flex; }
// або
.wrap { display: inline-flex; } // флекс-аналог inline-block

Ні для обгортки, ні для його дітей першого порядку не можна використовувати флоати (елементи автоматично отримують стиль display: block).

Далі задаємо напрямок розміщення дочірніх блоків. Є 4 варіанти напрямку:

.wrap { display: flex; flex-direction: row; }
.wrap { display: flex; flex-direction: row-reverse; }
.wrap { display: flex; flex-direction: column; }
.wrap { display: flex; flex-direction: column-reverse; }

row - горизонтальний напрямок потоку;
column - вертикальний;
reverse - змінити напрямок розташування на протилежний.

При застосуванні вертикального напрямку решта стилів застосовується так, ніби блок повернений на 90 градусів проти часової стрілки, поверніть голову на бік :)

Можна вказати чи флекс-обгортка буде багатострочною, чи однострочною:

.wrap { flex-wrap: nowrap; }
.wrap { flex-wrap: wrap; }
.wrap { flex-wrap: wrap-reverse; }

Попередні два стиля можна об'єднати одним:

.wrap { flex-flow: row nowrap; }
.wrap { flex-flow: column-reverse wrap; }

Розтягування контенту:

.wrap { justify-content: flex-start; }
.wrap { justify-content: flex-end; }
.wrap { justify-content: center; }
.wrap { justify-content: space-between; }
.wrap { justify-content: space-around; }

Розтягування контенту.

Вирівнювання контенту по вертикалі:

.wrap { align-items: flex-start; }
.wrap { align-items: flex-end; }
.wrap { align-items: center; }
.wrap { align-items: baseline; }
.wrap { align-items: stretch; }

Вирівнювання контенту по вертикалі.

Для багатострочного флекса є ще властивість, яка керує строками:

.wrap { align-content: flex-start; }
.wrap { align-content: flex-end; }
.wrap { align-content: center; }
.wrap { align-content: space-between; }
.wrap { align-content: space-around; }
.wrap { align-content: stretch; }

Вирівнювання строк у флексбоксі.

Дочірні елементи

Стиль, яким можна керувати порядком елементів:

.wrap .item:nth-child(1) { order: 2; }
.wrap .item:nth-child(2) { order: 3; }
.wrap .item:nth-child(3) { order: 1; }
.wrap .item:nth-child(4) { order: 4; }

Стиль, яким можна створити "сітку" з дочірніх елементів. Значення стилю - ціле чи дробне число більше нуля. Якщо задано нуль - блоки розтягуються відповідно іншим стилям.

.wrap .item { flex-grow: 2; }
.wrap .item:nth-child(2) { flex-grow: 2.5; }
.wrap .item.long { flex-grow: 4; }

flex-shrink - стиль, що дає можливість стискатися елементу, числове значення, по замовчуванню - 1.

flex-basis - розмір елемента "по замовчуванню" до його розтягування відповідно стилів flex.

flex - стиль, що об'єднує в собі flex-grow, flex-shrink та flex-basis:

.wrap .item { flex: 2 1 auto; }

align-self - дозволяє змінити стиль align-items батьківського елемента для конкретного дочірнього елемента:

.wrap .item.end { align-self: flex-end; }

Сітка

Наразі розробляється ще одна цікава концепція розмітки - сітка.

.wrap { display: grid; }

Браузери ще не підтримують цей стиль, його можна протестувати у браузерах, додавши вендорні префікси (а також, можливо, доведеться включити у браузері підтримку тестових стилів).

Орієнтовно - в 2017 році сітка увійде в користувацькі версії браузерів, а у 2018-2019 роках позбудеться вендорних префіксів.

Фактично, цей стиль витіснить сітки, такі як Bootstrap, Skeleton та інші.

Display-сітка має значні переваги перед тими сітками, які використовуються зараз:
- ви не обмежені заготовкою, кожен батьківський блок може мати свою власну сітку;
- вам не треба робити заготовки на всі можливі колонки, кожний батьківський блок з сіткою описується стилями "під себе";
- колонки можуть виражатися не цілими числами, наприклад: є 7-колоночний блок, а перша колонка в нього може бути 2,3 ширини (23%).

Сітки вам на самоопрацювання, гугл: "css display grid"
