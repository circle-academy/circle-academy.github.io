2.2. Налаштування Adobe Photoshop.

Зображення 2.2.1. Скріншот головного вікна Adobe Photoshop.

Для порізки спочатку нам необхідно трохи налаштувати фотошоп.

Першим чином включіть панель інструментів роботи з текстом:

Window > Character

Ми будемо використовувати панелі Layers, Character, Paragraph, решту панелей можна прибрати, щоб вони нас не відволікали і не займали простору екрану.

Виберіть інструмент Move Tool (зліва перша іконка зверху), поставте галочку "Auto-Select" і в списку виберіть Layer. Це дозволить кліком мишки активізувати той шар, по якому елементу ми клацнули.

Зображення 2.2.3. Налаштування інструменту Move Tool.

Налаштуйте використання одиниць вимірювання, встановивши значення Pixels в перших двох пунктах:

Edit > Preferences > Units & Rules...

Зображення 2.2.3. Налаштування одиниць вимірювання.

За потреби можна переналаштувати розміщення тимчасових файлів фотошопа:

Edit > Preferences > Performance...

2.7. Збереження зображень, типи файлів.

File > Save For Web...
Ctrl + Alt + Shift + S

Формати файлів:

JPG - для фотографій, великих фонових зображень;

PNG - для всіх елементів дизайну, іконок, елементів з прозорістю;

GIF - для дуже маленьких елементів з малою кількістю кольорів.

Імена файлам потрібно давати з розумом, таким чином, щоб подібні файли знаходилися поруч в списку файлів. Для цього використовуються префікси.

Формат архівів:

projectName_date.zip
futbolki_20151110.zip
futbolki_20151110a.zip
futbolki_20151110b.zip
futbolki_20151110c.zip

Для архівування використовуємо лише формат ZIP, всі сучасні операційні системи мають інструменти для розпакування zip-архівів. Архіви в форматах RAR, 7z можуть не розпакуватися у замовника.

Коли я перевірю ваше домашнє завдання - то зроблю файл-репорт з описом ваших помилок чи похвалою, що все добре і правильно зроблено:

futbolki_20151110.txt

Якщо ви допустили якісь незначні помилки і розумієте їх, як їх виправити і уникнути в майбутньому - то нову, виправлену версію на сервер можна не заливати.

Якщо наробили купу помилок, то роботу необхідно переробити, виправити і заляти на сервер з новою датою архіва.
